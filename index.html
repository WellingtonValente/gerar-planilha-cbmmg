Gerar Planilha Excel (.xlsx) ou CSV

Cole abaixo o conteúdo CSV ou JSON fornecido pelo seu GPT. Tudo acontece no seu navegador — nenhum dado é enviado a servidores.

Sobre o modo .xlsx: para funcionar sem erros em qualquer navegador, esta página tenta usar uma cópia local da biblioteca xlsx.full.min.js. Se ela não estiver presente, o botão .xlsx fica desativado e você pode usar o botão Baixar .csv (Excel abre normalmente). Para habilitar o .xlsx, basta fazer o upload do arquivo xlsx.full.min.js na mesma pasta deste index.html. Instruções no README do repositório.
